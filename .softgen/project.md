# Local News Source Discovery Platform

## Vision
A professional research tool that automatically discovers and categorizes ALL local news sources for any location. Users enter a country and zip code, and the system performs deep research to find businesses, organizations, community bodies, municipalities, schools, clubs, facilities, and authorities that publish local news.

## Design
Colors (HSL):
- `--primary: 215 25% 27%` (deep slate) - authority, professionalism
- `--accent: 174 72% 56%` (vibrant teal) - discovery, highlights
- `--muted: 215 16% 47%` (muted slate) - secondary text
- `--background: 40 40% 97%` (warm cream) - reduces eye strain
- `--foreground: 215 25% 15%` (dark slate)

Fonts:
- Headings: Plus Jakarta Sans (600, 700)
- Body: Work Sans (400, 600)

Style: Clean data-focused interface with progressive disclosure, smart categorization, and filterable results.

## Features
- Country + zip code search with deep crawling
- Automatic categorization by type (business, organization, government, education, etc.)
- Topic focus analysis and community importance scoring
- News frequency tracking
- Filterable, sortable results table
- Search history and saved sources
